# My Home Bar — Shopify Theme

A custom Shopify theme built for **My Home Bar**, a fictional cocktail and home
bartending brand, created as part of a Shopify development course and adapted
for portfolio purposes.

## About the Brand

My Home Bar brings the craft cocktail experience out of upscale bars and into
home kitchens. The concept centers on small-batch syrups, hand-selected
spirits, premium mixers, and curated cocktail bundles — aimed at people who
want bar-quality drinks without the bar.

## Tech Stack

- Shopify Online Store 2.0 (JSON templates, sections, blocks)
- Liquid, JavaScript, CSS
- Built and managed locally with Shopify CLI
